# Issue Tracker

This project uses **Trello** for issue tracking.

## Workflow

Cards follow the column flow: **Backlog → In Progress → Done**.

- New tasks are created as cards in the **Backlog** column.
- Work in progress moves to **In Progress**.
- Completed work moves to **Done**.

Since there is no CLI integration for Trello, when skills need to create or reference issues:
- Document them as markdown summaries in `.scratch/` for local reference.
- When creating new tasks, describe the card content (title, description, checklist) so it can be manually added to the Trello board.
